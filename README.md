# Tripleten web_project_api_full

A full-stack web application built with **React**, **Node.js**, **Express**, **MongoDB**, and **Jest**. 
This project serves as a comprehensive demonstration of my full-stack development skills acquired during the **TripleTen** bootcamp, featuring a robust single-profile CRUD system.

## 🛠️ Tech Stack

| Component | Technology |
| :--- | :--- |
| **Frontend** | JavaScript, React |
| **Backend** | Node.js, Express |
| **Database** | MongoDB |
| **Testing** | Jest |
| **Architecture** | RESTful API |

## ✨ Skills Demonstrated

*   **CRUD Mastery:** Implementing full Create, Read, Update, and Delete operations in a production-like environment.
*   **Database Design:** Designing clean, normalized MongoDB schemas for efficient data retrieval.
*   **Quality Assurance:** Writing reliable unit and integration test cases with **Jest**.
*   **UI/UX:** Building responsive, user-centric interfaces with **React** hooks and components.
